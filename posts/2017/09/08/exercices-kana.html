Publié le 08/09/2017 dans Japonais.

Après avoir vu les deux syllabaires japonais, nous allons faire quelques exercices !

Reconnaître les hiragana

Exercice 1

Consigne : retranscris les mots écris en rômaji avec les hiragana.

  • kaki
  • nori
  • sushi
  • anime
  • kokeshi
  • futon
  • tsunami
  • otaku
  • teriyaki
Afficher la correction

  • kaki = かき
  • nori = のり
  • sushi = すし
  • anime = あにめ
  • kokeshi = こけし
  • futon = ふとん
  • tsunami = つなみ
  • otaku = おたく
  • teriyaki = てりやき

Reconnaître les katakana

Petit rappel : les katakana sont principalement utilisés pour écrire en japonais les mots provenants de langues étrangères. Il est parfois utile de connaître du vocabulaire étranger, notamment anglais.

D’autre part, pour obtenir la meilleure ressemblance phonétique possible, certaines syllabes japonaises sont allongées à l'aide d'un tiret horizontal semblable à celui-ci : ー. En rômaji, ce symbole est traduit par une deuxième voyelle identique à celle du kana qui précède le tiret. On peut aussi remplacer cette deuxième voyelle par un accent circonflexe sur la voyelle de la syllabe allongée.

Exercice 3

Consigne : retranscris en rômaji les noms de pays et les noms communs écris en katakana. Si tu y arrives, traduis-les ensuite en français !

  • アメリカ
  • オーストラリア
  • フランス
  • イタリア
  • コーヒー
  • トマト
  • マウス
Afficher la correction

Correction :

  • アメリカ = amerika = America / Amérique
  • オーストラリア = ôsutoraria = Australia / Australie
  • フランス = furansu = France
  • イタリア = itaria = Italia / Italie
  • コーヒー = kôhî = coffee / café
  • トマト = tomato = tomate
  • マウス = mausu = mouse / souris

Exercice 4

Consigne : retranscris en katakana les mots écris en rômaji et trouves leur correspondance en anglais et en français.

  • monitâ
  • chîmu
  • kurisumasu
  • sutâto
  • kamera
  • hîrô
Afficher la correction

Correction :

  • monitâ = モニター = monitor / moniteur
  • chîmu = チーム = team / équipe
  • kurisumasu = クリスマス = Christmas / Noël
  • sutâto = スタート = start / départ
  • kamera = カメラ = camera / appareil photo
  • hîrô = ヒーロー = hero / héros
